# -football-unity-game-3d
A fun 2-character football game made in Unity! Control your character, compete with a friend, and try to score goals in each other’s nets. Simple physics, responsive controls, and local multiplayer make it a casual and engaging game.

Features
Two-player local gameplay: Play with a friend on the same keyboard or device.
Simple football mechanics: Move your character, kick the ball, and score goals.
Scoring system: Each goal updates the scoreboard for both players.
Interactive environment: The football field includes goalposts, boundaries, and a ball with realistic physics.
Built with Unity: Designed to help beginner and intermediate developers learn game development principles.

How to Play
Player 1 Controls: W (up), A (left), S (down), D (right)
Player 2 Controls: Arrow Keys
Objective: Score as many goals as possible while defending your own net.


This project is perfect for anyone looking to explore game development in Unity or to create a fun, local multiplayer experience. You can expand it further by adding animations, sound effects, or even online multiplayer. Overall, it demonstrates a mix of programming, game design, and problem-solving skills, making it an ideal project for both learning and showcasing your Unity abilities.
